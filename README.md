# weekly

[WecTeam 周刊](https://github.com/wecteam/weekly)是由京东社交电商部维C团维护的技术周刊，每周从前端同学阅读的技术文章中精选而来，每周五发布。

更多「原创」前端技术文章，欢迎关注微信公众号「WecTeam」。
